# 简介

根植于 Ruoyi、脱胎于 Ruoyi-Geek 的一个 Demo，感谢前两个项目的所有作者以及贡献者。主要是想学习一下 Jimmer，体验一下改造旧项目的快(痛)乐(苦)🤣

文档系统使用 [Docusaurus](https://docusaurus.io/) 😄

这个仓库移除了原框架较多的东西，属于破坏性更改，后续不再兼容前面所述的框架。因此，

这个项目适合于：

- 纯粹的学习
- Jimmer 学习
- Java 学习
- Spring Boot 学习
- Maven 学习
- Docusaurus 部署学习

## 变更说明

### 后端变更

包括但可能不限于：

- 移除扩展模块和一些包
- 移除 AjaxResult
- 移除 fastjson
- 移除 httpclient4
- 移除 jjwt，添加 java-jwt
- 移除 mybatis
- 移除 pagehelper
- 移除 knife4j（swagger）
- 移除 Anonymous
- 移除 quartz
- 移除 原框架提供的一些工具类（不详叙，比较多）
- 移除 AuthenticationContextHolder（重要，单独列出）
- 移除 Online 扩展
- 移除 代码生成器
- 移除 其它浪子认为不必要的封装（包括但不限于依赖、工具类等），忘了，不列出
- 引入 spring-boot-parent 管理项目
- 引入 Jimmer
- 接口、命名方式、结构组合、文件等按照自己的方式进行重命名
- 配置文件修改
- ...

### 前端变更

- 移除 在线表单设计器
- 工具版本升级
