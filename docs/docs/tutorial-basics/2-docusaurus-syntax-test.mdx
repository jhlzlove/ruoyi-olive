# Docusaurus 语法测试

## 代码块

### 默认

```rust
fn main() {
    println!("Hello, world!");
}
```

### 带 title

```rust title="main.rs"
fn main() {
    println!("Hello, world!");
}
```

### 带行号

```rust title="main.rs" showLineNumbers
fn main() {
    println!("Hello, world!");
}
```

### 块内行高亮

```rust title="main.rs" showLineNumbers
fn main() {
    // highlight-next-line
    println!("Hello, world!");
}
```
