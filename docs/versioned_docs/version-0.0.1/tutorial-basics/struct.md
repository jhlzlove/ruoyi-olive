# 模块

```bash
├── docs
├── olive-api
├── olive-base
├── olive-common
├── olive-framework
├── olive-plugins
├── olive-system
├── podman-compose.yml
├── pom.xml
├── sql
```

### 依赖关系

olive-base 是最基础的模块，包含所有实体

olive-common 模块与框架无关，可以单独抽离出去为任何业务框架使用

olive-framework 模块为项目的整体配置中心

olive-system 为系统模块，是本业务框架的核心功能点

olive-plugins 是插件模块
